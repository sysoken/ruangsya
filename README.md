# Ruangsya Personal Profile

This is the personal profile landing page for **Syaukani**, a Software Engineer with over 6 years of experience in enterprise application and software development. 

The website is designed with a bespoke, premium **Emerald, Amber, and Deep Onyx** dark theme with glassmorphism elements, micro-animations, and a fully functional Light/Dark mode toggle.

## Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop views.
- **Premium Aesthetics:** Glassmorphism UI, floating glowing backgrounds, and a high-tech grid.
- **Dynamic Interactivity:**
  - Typing effect in the hero section.
  - Custom cursor glow that interacts with elements on hover.
  - 3D tilt effects on cards.
  - Scroll progress bar.
- **Dark/Light Mode:** Seamlessly switch between the default dark theme and a clean light theme.

## Tech Stack

- **HTML5:** Semantic structuring.
- **CSS3:** Custom variables, Flexbox/Grid, and modern animations (Vanilla CSS, no frameworks).
- **JavaScript (Vanilla):** DOM manipulation, Intersection Observer for scroll reveal animations, and `localStorage` for theme preferences.

## Usage

Since this is a static website, you don't need to install any dependencies.

1. Clone the repository:
   ```bash
   git clone https://github.com/sysoken/ruangsya.git
   ```
2. Open `index.html` directly in any modern web browser.
